Sobre o Bubble Sort

O bubble sort é um dos algoritmos mais simples de ordenação. Em linhas gerais, ele consiste em percorrer a sequência e comparar os elementos adjacentes e, quando necessário, efetuar trocas entre eles, colocando o maior valor na sua posição correta. Para a execução do bubble sort são necessárias múltiplas passagens pelo vetor. No melhor caso, quando a sequência já encontra-se ordenada, sua complexidade é de O(n), onde n se refere a quantidade de elementos da sequência. Nesse caso, o algoritmo percorre a sequência apenas uma vez. No pior caso, a complexidade desse algoritmo é de O(n²).

O tempo gasto em uma ordenação utilizando esse método é proporcional ao quadrado do número de elementos de sua entrada. Por isso, não é recomendado usá-lo para sistemas que precisem de velocidade ou que lidem com uma grande quantidade de dados.
